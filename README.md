# Mon Journal

This is a personal diary app coded using [Expo](https://expo.dev)

## Get started

1. Install Bun

   ```bash
   curl -fsSL https://bun.sh/install | bash
   ```

2. Install dependencies

   ```bash
   bun install
   ```

3. Start the app

   ```bash
    bun start
   ```
